# musinsa_analysis

안녕하세요!  
본 저장소는 **소비패턴 분석 프로젝트**를 담고 있습니다.

---

## 📂 포함된 파일

- `musinsa_analysis.ipynb`  
  파이썬으로 무신사 주문내역(CSV)을 분석하고, 연도별·브랜드별 소비 패턴을 시각화한 주피터 노트북 파일입니다.

---

## 📌 주요 내용

- CSV 파일 불러오기 및 가공
- 리스트 메서드, 정규표현식, 반복문과 조건문 실습
- Pandas를 활용한 데이터 프레임 처리
- Matplotlib으로 시각화
- csv 모듈과 pandas 비교 정리
